## Bagaimana cara memulai?

Pertama setup dulu : 
- `npx specsmd@latest install`
- `npx @chorus-aidlc/chorus`

### SLDC Style Workflow

saat menggunakan Specs, 
kita bisa memilih pendekatan development atau a development flow:

- FIRE - Adaptive execution - Brownfield/monorepo ready, right-sizes rigor to complexity
- AI-DLC - Full methodology - Comprehensive traceability, DDD or Simple bolt types
- Simple - Spec generation only (Kiro style) - Creates requirement/design/task docs, no execution tracking
- Ideation - Creative ideation - Spark → Flame → Forge idea generation and shaping
